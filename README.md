# claudeESReviewer — ケース面接コーチ（MBB系）

Claude Code 上で **ケース面接 / フェルミ推定** の壁打ち練習ができるスキル群。
MBB系（マッキンゼー・BCG・ベイン水準）の戦略コンサル面接官として進行し、最後に **100点満点で総評** を返します。

## 何ができるか

- **フェルミ推定**の練習（市場規模・需要量推定など）
- **ビジネスケース**の練習（売上向上・利益改善）
- **市場参入・新規事業**判断の練習
- お題は **面接官（こちら）から出題** も **候補者（あなた）からの持ち込み** も両対応
- 進行中、MECE違反 / 論理の飛び / フォーカスのズレ / 大論点見失い を検知し、**答えを教えずに質問で誘導**
- セッション終了時、**5観点 / 100点満点** で採点。**MBB合格水準は80点以上**。

## 使い方

このリポジトリで Claude Code を起動し、次のように話しかけるだけです。

```
ケース面接の練習がしたい
```

または:

```
フェルミ推定をやって
ビジネスケース、売上回復のお題で
市場参入のケース、私が持ち込んだお題でやりたい
```

`case-interviewer` スキルが自動起動し、オープニングで以下4点を確認します。

1. **お題形式**: A. フェルミ / B. ビジネスケース / C. 市場参入
2. **出題の主体**: a. 面接官が出題 / **b. 候補者が持ち込み**（その場で1行でお題を提示）
3. 想定時間（任意・標準20〜40分）
4. 厳しさ（任意・デフォルトはMBB標準=厳しめ）

その後、お題提示 → 確認質問 → 構造化 → 深掘り → 最終回答 → 質疑応答 → **総評** の順で進行します。

## 終了と総評

セッションを終わるときは次のいずれかを伝えてください:

```
終了
ここまでで終わります
総評ください
```

`case-final-feedback` スキルが起動し、観点別配点で採点した総評が出ます:

| 観点 | 配点 |
|---|---:|
| 構造化力 | 25 |
| 深掘り・分析力 | 25 |
| コミュニケーション | 15 |
| 創造性・ビジネス感覚 | 20 |
| プレッシャー耐性・議論姿勢 | 15 |
| **合計** | **100** |

| スコア | 判定 |
|---|---|
| **80以上** | **MBB合格水準（Strong Hire）** |
| 70〜79 | 上位戦略ファーム合格圏 / MBBはボーダー |
| 60〜69 | 改善で射程圏 |
| 60未満 | 大幅な準備不足 |

## スキル構成

```
.claude/skills/
├── case-interviewer/              … 本体: 面接官ペルソナ・進行制御
│   └── SKILL.md
├── case-coaching-techniques/      … 指導テクニック（MECE/縦横/論理飛び検知）
│   └── SKILL.md
├── case-fermi/                    … フェルミ推定の方法論
│   ├── SKILL.md
│   └── cases.md                   … フェルミお題バンク10問（面接官専用）
├── case-business-strategy/        … 売上利益・市場参入のフレーム
│   ├── SKILL.md
│   └── cases.md                   … ビジネスケースお題バンク10問（面接官専用）
└── case-final-feedback/           … 総評ルーブリック（100点満点）
    └── SKILL.md
```

`case-interviewer` が本体で、進行中に必要に応じて他4つを自動参照します。

## お題バンクについて

`cases.md` 配下には**面接官専用の想定分解・つまずきポイント**が含まれます。面接官（Claude）はこれを**候補者に開示しない**運用です。練習の前に自分でこのファイルを読んでしまうとネタバレになるので、本番感覚で練習したい場合は中身を見ないようご注意ください。

## 注意事項

- **音声入力は未対応です**（チャット入力のみ）。将来的に Whisper 等の音声入力スキルを追加する余地はあります。
- お題バンクは10問ずつ用意していますが、候補者持ち込みのお題でも進行可能です。
- スキルは Claude Code が起動時に自動的に読み込みます。プロジェクトルートで `claude` を起動してください。
