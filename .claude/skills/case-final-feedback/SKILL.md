# 総評ルーブリック（100点満点）

ケース面接セッション終了時に**必ず**このルーブリックで総評を返す。配点・目標点・出力フォーマットを厳守。

## 配点（合計100点）

| # | 観点 | 配点 | 評価ポイント |
|---|---|---:|---|
| 1 | **構造化力** | 25 | MECEの徹底度、フレーム選択の的確さ、大論点の捉え方、ロジックツリーの粒度 |
| 2 | **深掘り・分析力** | 25 | 仮説の質、定量感、サニティチェック、議論の飛びがないか、検証ステップ |
| 3 | **コミュニケーション** | 15 | 結論ファースト、論点の明示、面接官への質問の的確さ、説明の簡潔さ |
| 4 | **創造性・ビジネス感覚** | 20 | 切り口の独自性、現実妥当性、So what、競合反応・リスク認識 |
| 5 | **プレッシャー耐性・議論姿勢** | 15 | 反論への対応、自分の意見の防御、面接官との対話姿勢、沈黙時の振る舞い |
|   | **合計** | **100** | |

## 目標点（MBB合格水準）

| スコア | 水準 | 判定相当 |
|---|---|---|
| **80点以上** | **MBB合格水準** | Strong Hire |
| 70〜79点 | 上位戦略ファーム合格圏 / MBBはボーダー | Hire 〜 Borderline |
| 60〜69点 | 改善で射程圏 | Borderline |
| 60点未満 | 大幅な準備不足 | No Hire |

**総評では必ず「目標 = 80点以上（MBB合格水準）」を明示し、現スコアとのギャップを示す。**

## 各観点の採点ガイド（配点に対する目安）

### 1. 構造化力（25点満点）

| 点数帯 | 状態 |
|---|---|
| 22〜25 | 大論点を即座に捉え、MECEなツリーを宣言。複数の切り口の中から根拠付きで選択。 |
| 17〜21 | 概ねMECE。ツリーは出るが切り口の根拠説明がやや弱い。 |
| 12〜16 | 構造化はあるが MECE違反 / 粒度バラバラ / 大論点とのズレが見える。 |
| 6〜11 | フレームを思いつかず時間を費やす / 単発の打ち手列挙で構造なし。 |
| 0〜5 | 構造化に至らない。 |

### 2. 深掘り・分析力（25点満点）

| 点数帯 | 状態 |
|---|---|
| 22〜25 | 仮説→検証→示唆を一気通貫。数値感覚あり、複数ルート検証、So what が出る。 |
| 17〜21 | 仮説と検証はあるが、サニティチェックが甘い。数値の桁感は概ね合う。 |
| 12〜16 | 検証ステップが省略されることあり。数値の根拠が弱い。 |
| 6〜11 | 仮説のまま結論に飛ぶ。数値が出ない、または桁が大きく違う。 |
| 0〜5 | 分析というより印象論。 |

### 3. コミュニケーション（15点満点）

| 点数帯 | 状態 |
|---|---|
| 13〜15 | 結論ファースト徹底。論点の宣言、状況整理、質問の的確さに優れる。 |
| 10〜12 | 概ね結論ファースト。たまに前置きが長い。 |
| 7〜9 | 結論が後回し、説明が冗長。確認質問が少ない／要点を外す。 |
| 4〜6 | 何を言っているのか辿りにくい。質問が散漫。 |
| 0〜3 | 議論が成立しない。 |

### 4. 創造性・ビジネス感覚（20点満点）

| 点数帯 | 状態 |
|---|---|
| 17〜20 | 独自の切り口、現実的かつインパクトのある示唆、競合反応とリスクを織り込み。 |
| 13〜16 | 妥当な打ち手は出るが、独自性が薄い。リスク言及あり。 |
| 9〜12 | 教科書的な打ち手のみ。リスク・競合反応が浅い。 |
| 5〜8 | 非現実的な打ち手、または既知の常識のみ。 |
| 0〜4 | 「広告を打つ」程度で止まる。 |

### 5. プレッシャー耐性・議論姿勢（15点満点）

| 点数帯 | 状態 |
|---|---|
| 13〜15 | 反論にも論理で対応し意見を防御。一方で素直に修正もできる。沈黙でも崩れない。 |
| 10〜12 | 反論で多少揺らぐが立て直せる。 |
| 7〜9 | 反論で自説をすぐ撤回 / 逆に頑なで議論にならない。 |
| 4〜6 | 焦り・沈黙で議論停止。 |
| 0〜3 | 議論姿勢が成立しない。 |

## 採点運用ルール

- **甘くしない**: 80点はMBB合格水準。簡単に超えさせない。実際の発言の質と照らして厳密に。
- **必ず具体的な発言を引用**する。抽象論で済ませない。引用は短く（1文以内）。
- **改善点には必ず「代替案」**（こう言えば加点だった）をセットで示す。指摘で終わらせない。
- **観点間で点数の整合性**を取る（構造化が低いのに深掘りが満点に近い、は通常おかしい）。
- 候補者が**フェルミお題**の場合は「創造性・ビジネス感覚」の配点を実質的に「ビジネス感覚 + 推定の創意」と読み替えて評価する。
- セッションが極端に短かった（1〜2ターンで終了等）場合は、その旨を総括に明記し、暫定スコアであることを断る。

## 呼び出されるタイミング

`case-interviewer` から以下の合図で呼ばれる:

- 「終わります」「終了」「ここまで」「総評ください」「タイムアップ」等
- 候補者が最終回答を出し、逆質問も終わった時点
- 議論が明らかに行き詰まり、かつ候補者が続行意思を示さない時

**総評を出さずにセッションを終わらせない**こと。
